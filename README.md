# Little God

一个以“创世实验”为核心的单人网页沙盒游戏。

玩家不是通过升级成为更强的神，而是通过投放物种、设定世界法则、有限干预与长期观察，回答不同的创世命题，并将独特世界记录进《创世之书》。

## 当前阶段

当前仓库处于 **MVP 阶段 0：可运行生态原型**。

本阶段只验证以下问题：

1. 投放不同物种后，世界是否会产生可观察的数量变化；
2. 捕食、觅食、繁殖和死亡是否形成清晰反馈；
3. 改变世界规则是否能显著改变生态结果；
4. 玩家是否愿意观察同一个世界继续演化。

## 运行方式

项目不需要安装依赖或构建。

- 直接打开 `index.html`；或
- 使用任意静态服务器托管仓库根目录。

推荐本地运行：

```bash
python -m http.server 8000
```

然后打开 `http://localhost:8000`。

## 项目文档

- `docs/MVP_SPEC.md`：MVP范围、规则和验收标准；
- `docs/CURRENT_TASK.md`：当前阶段唯一任务；
- `docs/HANDOFF.md`：已完成内容、已知问题和下一步；
- `AGENTS.md`：长期开发约束。

## 技术原则

- 原生 HTML、CSS、JavaScript；
- 无构建步骤，仓库根目录可直接部署；
- 模拟逻辑与渲染逻辑分离；
- 每个阶段只新增一个可验证的核心能力；
- 不在生态循环成立前加入文明、宗教、科技树或复杂美术。
