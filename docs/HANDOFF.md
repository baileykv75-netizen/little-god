# HANDOFF

## 当前阶段

Stage 2：食物链与代际闭环。

状态：**代码已实现并完成语法检查与无界面批量模拟，等待真实浏览器试玩。**

## 本阶段完成内容

### 捕食闭环

- 食草兽具有普通移动、短时冲刺、耐力消耗与恢复；
- 猎食兽具有巡逻、锁定、追逐、扑击、失败休息、进食和食腐状态；
- 捕猎成功率考虑双方状态与连续失败补偿；
- 成功捕猎后存在饱食冷却，不会连续瞬杀；
- 默认世界改为18只食草兽和2只猎食兽。

### 动物代际

- 增加幼体、成年、老年阶段及视觉差异；
- 自然死亡改为老年概率，不再精确到寿命立即删除；
- 两类动物繁殖需要成年配对、能量、季节、资源和种群条件；
- 小种群在仍有配偶时获得适度繁殖补偿；
- 界面显示两类动物的年龄结构。

### 植物与养分

- 枯草在春季加快分解，过量枯草会抑制新生长；
- 植物生长消耗肥力，枯草和尸体分解恢复肥力；
- 尸体不再直接制造种子；
- 秋季健康草地可向附近空地传播种子；
- 种子衰减、结籽、萌发和肥力收支重新调整。

### 任务与评分

新命题要求：

- 连续自主共存8年；
- 食草兽后代至少8只；
- 猎食兽后代至少2只；
- 成功捕食至少5次；
- 至少完成2次春季恢复；
- 最近2年没有重新投放动物；
- 共存期间种群不跌破最低线。

历史完成状态与当前生态状态分开显示。生态评分先判断食物网结构；仅剩植物时最高只能评价为“食物网崩解”。

### 反馈与诊断

- 分物种显示出生、被捕食、饿死和老死；
- 显示本年捕猎尝试、成功和成功率；
- 显示幼体、成年、老年数量；
- 世界纪事只记录首次捕食、首次猎食兽后代、濒危、灭绝、年度显著变化和任务完成；
- 诊断升级为v3，记录年龄结构、猎食兽行为状态、捕猎失败、任务条件和明确的开始/暂停/恢复操作；
- 修复累计指标年份始终为0的问题。

4. 植物是否出现过度积累或种子耗尽；
5. 任务完成时是否确实已经形成代际循环；
6. 当前生态崩溃后，历史成就与当前状态是否清楚区分；
7. 导出新版诊断报告并附上主观槽点。
