# HANDOFF

## 当前阶段

Stage 1：四季、草地生物量与生态恢复基础。

状态：**代码已提交，等待人工试玩与诊断报告。**

## 本轮为什么重构

Stage 0 诊断显示：即使玩家把植物堆到上限，植物仍在第一轮冬季附近归零；由于植物既是食物又是唯一繁殖来源，归零后无法在春季恢复，随后猎食兽和食草兽依次灭绝。

因此本轮没有继续微调旧参数，而是替换了植物和季节底层。

## 已完成

### 完整四季

- 1个游戏年由春、夏、秋、冬组成；
- 春季种子萌发；
- 夏季鲜草稳定生长；
- 秋季植物结籽并产生枯草；
- 冬季鲜草转为枯草，地下种子继续休眠；
- 关闭四季后，世界保持温和夏季规则。

### 草地斑块

每个草地斑块保存：

- 鲜草生物量；
- 枯草生物量；
- 地下种子库；
- 土壤肥力；
- 斑块位置和范围。

食草兽只减少生物量，不再一次进食就删除整个植物繁殖节点。鲜草为0但仍有种子或枯草时，界面显示“休眠待萌发”，不判定为彻底灭绝。

### 最小物质循环

- 动物饿死或老死后留下尸体；
- 捕食后留下少量残骸；
- 猎食兽在没有活体猎物时可以食腐；
- 尸体逐渐分解并提高附近草地肥力和少量种子储备。

### 基础种群限制

- 食草兽繁殖受到季节、能量、局部食物和承载压力限制；
- 猎食兽繁殖要求足够猎物比例；
- 冬季动物繁殖停止；
- 捕猎加入冷却和成功率，不再接触即必杀；
- 默认世界改为约11个草地斑块、10只食草兽和2只猎食兽。

### 观察与诊断

界面新增：

- 鲜草、枯草和种子库；
- 尸体数量；
- 本年食草兽/猎食兽出生数；
- 被捕食、饿死和老死数量；
- 本年新萌发生物量；
- 结合储备、近期趋势和猎物比例的生态风险提示。

诊断报告现在直接读取模拟内部状态，记录：

- 三类植物资源；
- 草地斑块数量与平均肥力；
- 动物与尸体数量；
- 当前年和累计出生死亡原因；
- 捕猎尝试与成功次数；
- 共存时间和生态判断。

## 新创世命题

> 让草地、食草兽和猎食兽连续经历3个完整年份。

选择3年是为了让第一次试玩能够至少看到三次春季复苏和三次冬季压力，不再要求等待抽象的300年。
