## 当前已知限制

1. 植物仍依赖 `state.patches` 和圆形 `patch.radius`，不是连续生态地表；
2. 种子与根系收支不完整，种子可能无限积累；
3. 当前草地无法形成可靠的连续啃食缺口和恢复带；
4. 行为模块仍可能直接读取真实目标属性；Observation、空间记忆、完整求偶与生命周期重构均未开始；
5. 综合战力仍参与部分内部行为判断；
6. 随机数尚未统一为可复现的 seeded RNG。

1. 远近缩放与拖动时地表是否仍连续、无明显格边；
2. 食草兽是否在实际位置造成可见但不过度的啃食缺口；
3. 根系、种子与季节变化是否产生可理解的恢复过程；
4. 2560 单元运行时是否保持流畅；
5. 暂停、速度、投放、选择、跟随、重置和诊断导出是否保持正常。
