# Little God — Genesis Specification v0.1

## 1. 项目原则

Genesis版本以六条规则作为不可破坏的底层约束：

1. **物理约束**：位置、距离、速度、能量、体力、季节和资源必须真实参与结果；
2. **局部信息**：动物只能使用感知范围内、带误差的观察和有限记忆；
3. **显式遗传**：后代由父母等位基因重组产生，不直接复制模板；
4. **连续性状**：速度、力量、感知、性格等使用连续数值，不使用简单标签；
5. **概率选择**：强壮个体拥有更高繁殖机会，但不保证必然获胜；
6. **可解释性**：重要结果可以追溯到环境、感知、选择、行为、繁殖和基因变化。

本版本仍保留草地—食草兽—猎食兽三层生态，但它们仅作为最初的生态功能类型，而不是永久固定物种。

## 2. 属性分类

### 2.1 形态与结构

| 属性 | 含义 | 主要影响 |
|---|---|---|
| 体重 bodyMass | 个体质量和体型的近似量 | 惯性、能耗、威慑、战力、速度代价 |
| 肌力 musclePower | 肌肉输出能力 | 捕猎、冲突、求偶展示、加速 |
| 体表防护 armor | 皮毛、骨骼和外部防护 | 受伤风险、战斗耐受 |
| 灵活性 agility | 转向、躲避和动作控制 | 移动速度、逃生、扑击、落点控制 |

### 2.2 生理与代谢

| 属性 | 含义 | 主要影响 |
|---|---|---|
| 耐力 endurance | 持续运动和恢复能力 | 冲刺持续时间、追逐、体力上限 |
| 代谢效率 metabolicEfficiency | 单位食物维持生命的效率 | 基础能耗、饥饿风险 |
| 抗寒 coldTolerance | 低温环境下维持状态的能力 | 冬季能耗和活动能力 |
| 寿命潜力 lifespan | 健康环境下的预期寿命 | 老年时间和代际长度 |
| 繁殖潜力 fertility | 生殖投入和恢复能力 | 繁殖概率、繁殖冷却 |

### 2.3 感知与认知

| 属性 | 含义 | 主要影响 |
|---|---|---|
| 视觉距离 visionRange | 可发现目标的视觉范围 | 食物、敌人和配偶观察 |
| 听觉距离 hearingRange | 对移动和警戒声的观察范围 | 危险发现、同类定位 |
| 嗅觉距离 scentRange | 对食物、猎物和同类气味的范围 | 追踪和觅食 |
| 判断精度 perceptionAccuracy | 对目标状态估计的准确程度 | 择偶、风险判断、猎物选择 |
| 记忆跨度 memorySpan | 信息保留时间 | 后续路线、区域记忆与经验系统 |

### 2.4 行为与性格

| 属性 | 含义 | 主要影响 |
|---|---|---|
| 社会性 sociality | 接近、容忍和依赖同类的倾向 | 群居、配偶范围、未来群体系统 |
| 攻击性 aggression | 主动冲突和冒险倾向 | 捕食、竞争和求偶冲突 |
| 谨慎 caution | 回避未知风险的倾向 | 逃跑距离、路线保守程度 |
| 好奇 curiosity | 探索陌生区域的倾向 | 游荡和迁徙探索 |
| 择偶选择性 mateSelectivity | 对配偶质量差异的敏感程度 | 强壮异性的繁殖优势强度 |

### 2.5 超凡潜能

| 属性 | 含义 | 当前作用 | 后续作用 |
|---|---|---|---|
| 灵能容量 arcaneCapacity | 可储存的超凡能量 | 纳入潜在战力 | 技能消耗和强度 |
| 灵能稳定 arcaneStability | 控制超凡能力的稳定程度 | 属性展示 | 技能失败、冷却和反噬 |

Genesis v0.1不加入实际超凡技能，只建立其遗传属性和接口。

## 4. 属性相关性

初始种群不允许将所有性状独立抽取。当前使用潜在因子生成相关性：

- 体型越大，通常肌力和防护越高，但灵活性与代谢效率承受代价；
- 高运动能力同时提高灵活性、肌力和耐力；
- 高生命力提高耐力、代谢效率、抗寒和寿命；
- 视觉、听觉、嗅觉和判断精度共享感知因子；
- 社会性、攻击性、谨慎和好奇具有独立遗传方向；
- 寿命与繁殖速度存在轻微负相关；
- 灵能容量与灵能稳定共享超凡潜能因子。

该模型保证初始总体近似正态，但代际之后不强制保持正态。选择可能使均值、方差、偏度和峰数自然变化。

## 5. 遗传模型

### 5.1 数量性状

每项连续属性由4个简化基因位点控制；每个位点有父系和母系两个等位基因。

创始个体：

- 先根据相关正态分布生成遗传目标值；
- 再把该遗传效应分配到8个等位基因；
- 额外生成一次固定的发育环境偏差。

后代：

1. 每个位点从母本随机取得一个等位基因；
2. 从父本随机取得一个等位基因；
3. 每个等位基因有1.8%的小幅突变概率；
4. 后代遗传值由全部等位基因效应相加；
5. 表现值再叠加独立的发育环境影响。

因此两个表现一般的亲本仍可能通过重组产生更强或更弱的后代。

### 5.2 遗传率

不同属性使用不同遗传率。形态、运动和超凡潜能约0.64—0.72；感知约0.48—0.58；性格和繁殖约0.42—0.56。剩余差异由发育环境产生。

### 5.3 血脉

v0.1所有创始生命暂时属于100%“原初血脉”。血脉结构保存为双倍体标记位点。未来加入新血脉后：

- 纯合父母可以稳定传递完整血脉；
- 杂合父母可产生更纯、更杂或不表现该血脉的后代；
- 血脉祖源比例、关键位点纯合度和技能表达强度分开计算。

## 6. 表现型和实际参数

基因型不会直接作为移动速度。系统先生成表现型，再映射到模拟参数：

- 灵活性、耐力和体重共同决定步行与爆发速度；
- 体重和代谢效率共同决定能量上限及基础消耗；
- 耐力和代谢效率共同决定体力上限与恢复；
- 视觉、听觉和嗅觉共同生成综合感知半径；
- 谨慎影响危险反应范围；
- 社会性和综合感知影响局部配偶搜索范围；
- 寿命潜力直接影响老年阶段和死亡风险；
- 繁殖潜力修正繁殖概率和冷却时间。

## 7. 战力系统

### 7.1 基础战力

基础战力是供玩家观察的综合指标，由以下属性加权：

- 肌力23%；
- 灵活性17%；
- 耐力15%；
- 体表防护10%；
- 判断精度10%；
- 体型10%；
- 攻击性8%；
- 灵能容量7%。

### 7.2 当前战力

当前战力在基础战力上继续考虑：

- 当前能量；
- 当前体力；
- 幼体、成年或老年阶段。

### 7.3 情境结果

战力不是胜负判定的唯一输入。捕猎继续考虑：

- 双方当前战力差；
- 猎物体力；
- 距离；
- 捕猎失败补偿；
- 随机波动。

动物本身不能读取战力数字；战力只属于上帝洞察界面。

## 8. 局部感知与择偶

动物不能从全世界搜索最佳配偶。

繁殖流程：

1. 雌性个体满足成年、能量和繁殖冷却条件；
2. 仅在自身配偶感知范围内搜索可繁殖雄性；
3. 根据可观察健康、体型、运动状态、既有后代和带距离误差的判断形成吸引力；
4. 选择性越高，越倾向高吸引力个体；
5. 选择使用概率软选择，强者优势明显但不是必然获胜；
6. 双亲等位基因重组形成后代。

食草兽还受到局部植物资源限制；猎食兽还受到局部猎物数量与近期进食限制。

---

## 9. 植物基础结构

每块草地当前保存：

- 鲜草；
- 枯草；
- 根系生物量；
- 种子库；
- 土壤肥力；
- 谱系ID和植物代数。

根系维持已有草地春季复苏；种子主要用于根系受损斑块和新区域建立。食草兽啃食鲜草会造成少量根系损伤。

植物的完整数量性状遗传放在后续阶段，本版本只建立根系与谱系字段。

## 10. 时间尺度

| 速度 | 每季现实时间 | 每年现实时间 |
|---|---:|---:|
| 1× | 约20秒 | 约80秒 |
| 4× | 约5秒 | 约20秒 |
| 12× | 约1.7秒 | 约6.7秒 |

每个季节显示初期、中期、后期和进度条。速度改变只改变固定模拟步的执行数量，不改变生态规则。

## 11. Genesis v0.1可操作目标

1. 初始食草兽和猎食兽的属性形成有均值、有离散度的分布；
2. 点击“观察个体”后可以查看性别、代数、战力、感知和完整属性；
3. 同一物种不同个体的属性明显不同；
4. 动物只能在局部感知范围内选择异性；
5. 配偶选择对强壮个体有概率优势，不是全局最优和必然选择；
6. 后代记录父母、代数、基因型和表现型；
7. 后代可能超出父母某些属性的中间值；
8. 个体属性真实影响速度、能量、感知、寿命、捕猎和繁殖；
9. 1×速度下四季足够观察；
10. 诊断报告包含种群属性均值与标准差、遗传后代数和局部择偶次数。

## 12. 本版本明确不做

- 实际超凡技能；
- 新血脉混合；
- 群体和独狼组织；
- 个体长期记忆与学习；
- 求偶战斗和领地；
- 完整物种形成；
- 大地图、缩放和分层模拟；
- 玩家基因编辑器；
- 文明、宗教和科技。

这些功能只能建立在本版属性和遗传结构通过试玩验证之后。
