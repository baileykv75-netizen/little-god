# CURRENT TASK — Stage 0

## 任务名称

可运行生态原型与第一次人工试玩。

## 当前状态

代码实现完成后，本阶段停在“等待人工试玩”状态，不继续进入文明、进化或图鉴系统。

## 本阶段唯一目标

完成一个无需构建、可直接部署的网页原型，让玩家能够：

1. 投放萌芽、食草兽和猎食兽；
2. 观察觅食、捕食、繁殖、衰老、饥饿和灭绝；
3. 调整资源繁盛度、繁殖倾向和严酷季节；
4. 暂停、加速和重置世界；
5. 尝试完成“连续共存300年”的创世命题；
6. 从统计与世界纪事中理解生态变化。

## 必做文件

- `index.html`
- `styles/main.css`
- `src/main.js`
- `README.md`
- `AGENTS.md`
- `docs/MVP_SPEC.md`
- `docs/HANDOFF.md`

## 实现要求

- 使用原生HTML、CSS和JavaScript；
- 使用Canvas绘制世界；
- 无外部依赖；
- 页面根目录可以直接静态部署；
- 实体数量受控；
- 模拟速度变化不破坏规则一致性；
- 重要状态通过UI和世界纪事同步显示。

## 禁止事项

本阶段禁止加入：

- 可编辑基因；
- 物种随机突变；
- 文明、城市、宗教和科技；
- 神迹技能树；
- 多地图和存档；
- 战斗装备；
- 外部美术素材；
- 服务器或联机功能。

## 人工试玩清单

完成后请人工验证：

1. 页面是否一打开就知道如何投放生命；
2. 三种生命是否一眼可区分；
3. 1×速度下是否能看清追逐和进食；
4. 12×速度下是否能快速看到生态兴衰；
5. 调高资源繁盛度是否明显增加萌芽；
6. 调高繁殖倾向是否明显增加动物波动；
7. 开启严酷季节后冬季是否更危险；
8. 猎食兽过多是否可能造成食草兽灭绝；
9. 灭绝后重新投放是否能恢复实验；
10. 是否至少有一次产生“再观察一会儿”的欲望。

## 停止条件

满足代码验收后立即停止开发，更新 `docs/HANDOFF.md`，等待玩家反馈以下三件事：

- 哪个过程最有趣；
- 哪个过程看不懂；
- 哪个过程最无聊。
