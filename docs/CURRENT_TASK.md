# CURRENT TASK — Stage 2

## 任务名称

食物链与代际闭环修复。

## 当前状态

代码实现完成后停在“等待人工试玩与诊断报告”状态，不继续加入复杂地形、物种编辑、文明或神力系统。

## 本阶段唯一目标

让默认世界中的草地、食草兽和猎食兽形成可观察、可解释、能够延续多个世代的动态闭环：

1. 猎食兽能够锁定、追逐、扑击并成功捕食；
2. 食草兽能够逃跑，但不能永久以同速逃脱；
3. 两类动物具有幼体、成年和老年阶段；
4. 繁殖需要成年个体、配偶、能量、季节与资源条件；
5. 植物、枯草、种子和肥力形成有收入也有支出的循环；
6. 任务必须验证捕食、两类动物后代、春季恢复和至少8年自主共存；
7. 生态评分必须先判断食物网结构，动物全灭时不得报“健康”；
8. 诊断报告能区分两类动物的出生、死亡、年龄结构和捕猎失败。

## 明确不做

- 复杂地形和地图编辑；
- 基因、突变与物种特征编辑器；
- 神力资源和技能树；
- 文明、宗教、科技与战争；
- 多地图、账号、服务器和联机；
- 大型历史图表。

## 人工验收重点

1. 默认世界前2年是否能看到成功捕食；
2. 追逐、扑击、失败休息是否能被肉眼理解；
3. 前5年内猎食兽是否产生后代；
4. 20年运行后大多数随机世界是否仍保留三层生态；
5. 动物灭绝时评分和任务文案是否准确；
6. 右侧年龄结构是否能提前暴露断代风险；
7. 世界纪事是否只保留关键事件，而不被四季文本刷屏；
8. 导出的诊断是否能解释捕猎、代际和资源问题。

## 停止条件

完成代码与文档后停止开发，等待玩家运行至少20个游戏年并上传新版诊断报告。
