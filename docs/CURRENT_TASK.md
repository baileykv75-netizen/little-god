# CURRENT TASK — Genesis v0.1

## 任务名称

个体属性、局部感知、战力展示与显式遗传地基。

## 本阶段唯一目标

在现有三层生态原型上建立第一版演化数据结构，使每只动物成为具有独立基因型、表现型、性别、战力、感知和性格的个体，并让繁殖由局部感知范围内的异性选择和双亲基因重组产生。

## 必须完成

1. 建立形态、生理、感知、性格和超凡潜能五类属性；
2. 为食草兽和猎食兽定义均值、标准差、范围与遗传率；
3. 创始种群按相关近似正态分布生成；
4. 每项数量性状使用简化双倍体多位点基因；
5. 后代从双亲重组等位基因，并加入小概率微突变和发育环境影响；
6. 动物繁殖必须区分性别，且只搜索局部感知范围内的异性；
7. 更强壮的候选配偶获得更高概率，但不能保证必然被选中；
8. 属性实际影响移动、能量、体力、感知、寿命、繁殖和捕猎；
9. 增加上帝个体观察面板和种群均值/标准差；
10. 1×下每季约20秒，并显示季节阶段与进度；
11. 诊断包含遗传和属性分布信息；
12. 新增 `docs/GENESIS_SPEC.md`。

## 明确不做

- 实际超凡技能；
- 血脉混合和技能表达；
- 群居、独狼、首领和群体技能；
- 完整记忆、学习和领地；
- 物种分化；
- 大地图与缩放；
- 上帝基因编辑。

## 验收重点

1. 重置世界后猎食兽必有一雌一雄；
2. 观察不同个体时属性存在合理差异；
3. 种群属性均值接近设定基础值，标准差不为0；
4. 后代显示第1代及父母记录；
5. 后代属性由双亲重组，不是固定模板；
6. 配偶距离超过感知范围时不能繁殖；
7. 战力和感知数值真实影响模拟；
8. 页面无致命错误，原有暂停、速度、投放和重置仍可使用。

## 停止条件

实现代码、文档和基础自动检查后停止，等待人工试玩和新版诊断。
